# PolicyPulse AI
### Agentic Citizen Voice and Policy Intelligence Platform

A multi-agent NLP system that analyzes citizen complaints about government policies and generates structured intelligence briefs for legislators.

## Agents
- Agent 0: Content Moderation
- Agent 1: Voice Transcription
- Agent 2: Issue Classifier
- Agent 3: Sentiment and Stance
- Agent 4: Report Generator

## Run Locally
pip install -r requirements.txt
python3 app.py

## Stack
- Python, Gradio, HuggingFace Transformers
- Custom Python agent orchestration
- Whisper ASR, MuRIL, BERTopic
